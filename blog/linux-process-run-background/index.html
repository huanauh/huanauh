screen 命令

创建 screen

$ screen -S test

保留 screen

$ ctrl+a+d

按ctrl+a，然后再按d即可保留 screen

查看 screen

$ screen -ls

恢复 screen

$ screen -r test

nohup 命令

后台运行

$ nohup server -c ./server.josn &

示例命令为server -c ./server.josn

后台运行并重定向标准输出

$ nohup server -c ./server.josn 2>&1 &

停止后台运行

$ eval $(ps -ef | grep "server" | awk '{print "kill "$2}')
