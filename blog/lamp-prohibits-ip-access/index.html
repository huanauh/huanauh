LAMP 禁止 IP 直接访问

Han Leon

January 16, 2019

针对开启了虚拟主机的 LAMP，修改conf/extra/httpd-vhosts.conf文件，添加以下内容：

<virtualhost 1.1.1.1:80>  #1.1.1.1为你的 IP
ServerName 1.1.1.1
<Directory />
    Order Allow,Deny
    Deny from all
</Directory>
</VirtualHost>

没有开启虚拟主机的 LAMP 直接修改httpd.conf文件。 需要注意的是，原.conf文件中的ServerName或ServerAlias项要修改你的域名，保证你的域名可以正常访问。
