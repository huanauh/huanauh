Crt 与 key 格式证书转换为 pem 格式

驩讙懽歡   收录于 Linux教程
2018-10-23  约 128 字  预计阅读 1 分钟   次阅读  

执行

1

$ cat test.key test.crt > test.pem

即可转换为 pem 格式。

  • 注：因 pem 文件将 cer 和 key 文件内容合并，文件内容中END PRIVATE KEY与EBEGIN CERTIFICATE分别占用一行。不同 Web server 对 pem 文件内容格式要求不甚相同，可能需要将END PRIVATE KEY与EBEGIN CERTIFICATE合并在同一行。
