Linux 进程后台运行的方法

Leon   收录于 Linux教程
2018-09-16  约 116 字  预计阅读 1 分钟 

screen 命令

创建 screen

1

$ screen -S test

保留 screen

1

$ ctrl+a+d

按ctrl+a，然后再按d即可保留 screen

查看 screen

1

$ screen -ls

恢复 screen

1

$ screen -r test

nohup 命令

后台运行

1

$ nohup server -c ./server.josn &

示例命令为server -c ./server.josn

后台运行并重定向标准输出

1

$ nohup server -c ./server.josn 2>&1 &

停止后台运行

1

$ eval $(ps -ef | grep "server" | awk '{print "kill "$2}')
