修复 LookingGlass 在 PHP7 上无法使用的问题

驩讙懽歡   收录于 Linux教程
2018-12-03  约 61 字  预计阅读 1 分钟   次阅读  

LookingGlass 在 PHP7 环境无法使用，解决方案如下： 在根目录下ajax.php文件中找到以下内容

1
2

// execute command
$output = $lg->$_GET['cmd']($_GET['host']);

修改为

1
2

// execute command
$output = $lg->{$_GET['cmd']}($_GET['host']);

即可。
