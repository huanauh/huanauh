运行 bat 脚本时隐藏 cmd 窗口

驩讙懽歡   收录于 Windows教程
2018-12-18  约 50 字  预计阅读 1 分钟   次阅读  

在需要运行的 bat 脚本前添加以下内容：

1
2
3
4
5

@echo off
if "%1" == "h" goto begin

mshta vbscript:createobject("wscript.shell").run("""%~nx0"" h",0)(window.close)&&exit
:begin

REM

即可。运行时 cmd 窗口会闪烁一下。
