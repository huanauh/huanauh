Lighttpd 强制 Https 访问

驩讙懽歡   收录于 Linux教程
2018-10-24  约 52 字  预计阅读 1 分钟   次阅读  

编辑lighttpd.conf文件，添加以下内容

1
2
3
4
5
6

#force https
$SERVER["socket"] == ":80" {
    $HTTP["host"] =~ "([^:/]+)" {
        url.redirect = ( "^/(.*)" => "https://%0:443/$1" )
    }
}

即可。
